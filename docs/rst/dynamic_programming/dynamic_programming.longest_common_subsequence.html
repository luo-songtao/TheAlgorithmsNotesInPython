dynamic_programming.longest_common_subsequence.lcs_length(s1, s2)[源代码]¶
逐个遍历S1,S2所有元素：

  1. 若发现相同元素s_i=s_j，那么子序列S1[:i]和S2[:j]的lcs长度应该是子序列S1[:i-1]和S2[:j-1]的lcs长度值+1

  2. 否则S1[:i]和S2[:j]的lcs长度必定等于S1[:i-1]和S2[:j]或者S1[:i]和S2[:j-1]的lcs长度

可见这种情形也具备

最优子结构：S1[:10]和S2[:15]的lcs最优解包含了子问题S1[:8]和S2[:15]的lcs最优解 重叠子问题：S1[:10]和S2[:15]和S1[:12]和S2[:18]同时包括了如S1[:8]和S2[:8]的lcs的子问题

不过这里的动态规划问题是在根据条件排除子问题
