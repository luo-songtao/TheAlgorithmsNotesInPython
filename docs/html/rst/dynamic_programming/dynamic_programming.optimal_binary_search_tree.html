dynamic_programming.optimal_binary_search_tree module¶

Author: luo-songtao 最优二叉搜索树： 每个节点的信息被搜索的概率不一，被搜索概率越高的节点信息应当离根节点越近，这样搜索的代价会更低 （深度+1）* 节点概率 = 该节点的搜索代价
